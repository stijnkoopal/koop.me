# koopal.io
Source code for my [personal website](https://koopal.io).

## Install
```bash
yarn
```

## Develop
```bash
yarn dev
yarn lint
yarn storybook
```


## Test
```bash
yarn test
```

## Build
```bash
yarn build
```

## Other commands
Analyze bundles
```bash
yarn build analyze
```

Dependency management
```bash
yarn dependencies
yarn dependences:upgrade
# On error, run first:
# rm -rf node_modules && yarn
```
